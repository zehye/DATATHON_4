# DATATHON_4

## 1. 핵심 요약

본 연구는 신한카드 영세/중소 가맹점의 고객 구성과 상권의 변동성이 매출과 폐업에 직접적인 영향을 미친다는 사실을 정량적으로 확인하였다. 
이러한 분석을 기반으로, 고객 구성 변화와 상권 변동성을 결합하여 향후 매장의 성장성을 동적으로 예측하는 **미래 온도 모델을 제안**한다.


## 2. 추진 배경 및 필요성

### 2.1 현황 및 문제 제기

신한카드 자영업자 플랫폼 MySHOP Partner는 매출 지표 중심의 분석에 치우쳐져 있으며, 고객 구성 변화나 상권 변동성이 폐업 리스크 또는 경영 건강도에 어떻게 영향을 미치는지에 대한 정량적 설명이 부족하다.

### 2.2 문제 정의

실제 소상공인은 단순한 매출 확인을 넘어 다음과 같은 근본적인 질문에 대한 답을 원한다.

- “내 가게가 지금 안정적인 단계인가?”
- “다른 가게에 비해 얼마나 위험한 상황인가?”
- “현재의 고객 구조가 미래에는 어떻게 변할 것인가?”

하지만 현재 MySHOP Partner 시스템에는 이를 직관적으로 진단하고 보여줄 수 있는 지표가 부재하다.

## 3. 연구 목표 및 방안

### 3.1 분석 목적

본 연구의 목적은 신한카드 데이터를 활용하여

① 고객 구성(신규/재방문/유동인구)과

② 상권 변동성(개업/폐업 패턴)을 통합한

정량적 경영 지표인 온도 지표(Temperature Index)를 정의하고, 이를 기반으로 선형 모델 기반의 미래 온도 예측 프레임워크를 구축하는 것이다.

### 3.2 분석 절차

본 연구는 다음 절차로 진행된다.

1. EDA를 통해 폐업 매장의 구조적 특징 규명
2. 고객 구성 → 매출 → 폐업 간의 선형 관계 증명
3. 상권 변동성을 정량화
4. 고객 구성, 상권 변동성의 가중치 산출
5. 고객 구성, 상권 변동성을 반영한 온도 예측 모델 설계 (ML)

## 4. 기대 효과

이를 통해

- 고객 구성 변화가 매장의 성장에 어떤 영향을 미치는지 규명하고
- 상권의 변동성을 반영한 위험 탐지 체계를 마련하여
- 향후 가맹점 단위의 경영 진단 및 맞춤형 컨설팅의 기반을 제공한다.
